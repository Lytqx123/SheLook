# SheLook｜跨境电商视觉运营决策系统

> 文档基线：2026-07-23。产品介绍以当前模块化单体、异步 Worker 和企业数据闭环基座为准。

## 一句话介绍

SheLook 面向跨境电商运营团队，将商品视觉策略、AI 素材生产、质量审核、效果预测、A/B 实验、经营指标回流和策略复盘，整合为一套以“视觉运营活动”为单位的经营决策系统。

> 文档状态：已按 2026-07-23 本地 development 部署复核更新。本文介绍已落地的产品工作方式；外部生成、平台同步和真实经营结果仍取决于已授权的第三方服务与业务数据。业务 API 凭据由管理员在网页配置，后端按租户加密保存。

## 产品定位

跨境电商的商品视觉往往同时影响点击、转化、退货和品牌表达，但选图、审图、投放和复盘常分散在多个工具与角色之间，最终仍依赖个人经验。SheLook 的目标不是替代运营人员做决定，而是让每一次视觉选择都能被说明、验证和沉淀：为什么推荐这套素材、适合哪个市场、可能改善什么指标、有哪些风险、是否应该进入实验、实验结果又该如何影响下一次策略。

## 核心工作方式

系统以“视觉运营活动”而非零散功能模块作为工作主线。运营人员先建立活动的商品、目标市场和经营目标，再推进视觉策略、素材生成、质量审核、效果预测、A/B 实验和学习复盘。活动页面会聚合候选素材、审核状态、预测结果、实验进展、真实指标、时间线与洞见，减少用户在商品、素材、审核和分析页面之间来回切换。

## 关键产品能力

**智能内容生产**：围绕商品和市场完成商品建档、视觉方案推荐、相似案例检索、图片与视频生成，并支持按平台规格导出素材。生成能力可接入外部模型服务，也保留本地或开发环境的降级方式。

**多层质量门禁**：素材进入后可经过自动检查与人工审核，记录合规、技术质量、审美与商业可用性等结果。高风险或不确定内容不会被包装为“已通过”，而是进入可处理的审核队列并保留原因。

**可解释的经营判断**：预测模块用于比较候选素材的 CTR、爆款潜力和退货风险，并在界面中呈现判断依据、置信度、风险提示和建议动作。它服务于运营决策和实验优先级，不替代人工的经营判断。

**实验驱动的验证机制**：运营人员可以从候选素材发起 A/B 实验，结合流量、点击和实验结果比较视觉方案。活动详情把实验状态与前序审核、预测结果放到同一条经营链路中，方便复盘“预测是否被验证、策略为何胜出或失败”。

**可感知的数据飞轮**：经营数据、审核结果和实验结论能够沉淀为活动洞见，并与样本同步、模型训练和版本管理相连接。用户看到的不只是一次活动的结果，还能看到哪些策略被验证、哪些风险被识别、哪些经验可用于下一次运营。

**角色化协同与治理**：运营、审核、分析、供应商和管理者拥有不同的首页重点与可见功能。日常界面优先呈现待审核素材、异常任务、运行中的实验和下一步行动；模型、审计、公平性和深度分析等专家能力被归入治理和分析层，避免挤占日常路径。

## 业务价值

SheLook 将“凭经验选素材”升级为“基于策略、预测、实验和真实反馈做决策”：它把分散的内容生产、审核、投放验证和复盘流程变成可追踪的运营链路；把高表现素材和被验证的策略沉淀为组织资产；让运营、审核、供应商和管理者围绕统一的数据口径协作。对于团队而言，价值不在于多生成一张图片，而在于持续降低视觉试错成本、提高协作效率，并让视觉内容逐步成为可经营的数据资产。

## 当前使用边界

平台已具备多租户、角色权限、审计、异步任务和数据回流的产品基础，但具体效果仍取决于外部平台接入、真实经营数据和组织流程。预测结果是决策辅助而非业绩承诺；合规和公平性分析是运营治理工具，不替代法律、平台政策或人工审核。
